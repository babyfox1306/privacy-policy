# Privacy Policy

Simple, generic privacy policy page for mobile apps.

## Setup GitHub Pages

1. Create a new repository on GitHub (e.g., `privacy-policy`)
2. Push these files to the repository
3. Go to Settings → Pages → Source: `main` branch
4. Your privacy policy will be available at: `https://TuanAnhNguyen.github.io/privacy-policy/`

## Files

- `index.html` - Privacy policy page
- `.nojekyll` - Required for GitHub Pages

## Usage

Update the contact email in `index.html` before using.
